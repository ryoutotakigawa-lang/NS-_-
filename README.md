# SNS Marketing Scheduler

SNS投稿のスケジュール管理、音楽リリースに向けたカウントダウン、そして **Gemini 2.5 Flash** を使ったAIベースの英語ローカライズ機能を備えた、音楽アーティスト・クリエイター向けマネジメントアプリです。

## 主な機能
- 📅 **週間スケジュールの管理** (日本時間や英語圏時間などの投稿推奨時間を表示)
- ⏳ **リリースカウントダウン** (リリース日を設定すると自動的に戦略タスクを作成)
- 🚀 **Buffer連携 / 枠数管理** (無料枠の範囲内でエクスポート可能)
- #️⃣ **ハッシュタグプリセット** (よく使うタグの保存とワンクリックコピー)
- 📊 **分析メモ** (投稿のインサイトや反響を記録)
- 🌍 **AI ローカライズ** (Google Gemini 2.5 Flashが日本語のキャプション文やメモを海外向けに最適化して翻訳)

## 必要な初期設定（APIキーの設定）

AIローカライズ機能を使用するには、Googleの生成AIである Gemini のAPIキーが必要です（無料で取得・利用できます）。
第三者がソースコードからあなたのキーを盗み見られないよう、当アプリでは「ユーザー自身がブラウザ上の設定画面からキーを入力する」安全な方式を採用しています。

### 1-1. Google AI StudioでAPIキーを取得する
1. [Google AI Studio (https://aistudio.google.com/app/apikey)](https://aistudio.google.com/app/apikey) にアクセスし、Googleアカウントでログインします。
2. 左上の「**Create API key**」をクリックします。
3. プロジェクトを選択（または新規作成）してAPIキー（`AIza...`から始まる文字列）を生成します。
4. キーをコピーして、誰にも教えずに手元に控えてください。

### 1-2. アプリ内に入力する
1. アプリを起動し、右上の「⚙️ **設定**」タブを開きます。
2. 「**Gemini APIキー**」の入力欄に、先ほど取得したキーを貼り付けます。
※キーはご利用のブラウザ内部（localStorage）にのみ暗号化されずに保存され、外部サーバーには一切送信されないため安全です。

## ローカル環境での起動方法

ご自身のPC（ローカル環境）で開発・カスタマイズを行うための手順です。
前提として、[Node.js](https://nodejs.org/ja/) がインストールされている必要があります。

1. ターミナルでこのプロジェクトのフォルダに移動します。
2. 以下のコマンドを実行して、必要なパッケージをインストールします。
   ```bash
   npm install
   ```
3. 続いて、以下のコマンドで開発サーバーを起動します。
   ```bash
   npm run dev
   ```
4. ターミナルに表示されたローカルURL（例: `http://localhost:5173`）をブラウザで開くと、アプリを利用できます。

## デプロイ（一般公開）について

このプロジェクトは Vercel や GitHub Pages などの無料ホスティングサービスと連携しやすいように設定されています（`vite`を使用）。
GitHub上でリポジトリを作成してプッシュした後、Vercel等と連携すれば数十秒で誰でもアクセス可能なアプリとして公開できます。APIキーもハードコードされていないため、そのまま公開リポジトリとしてアップロードしても安全です。
